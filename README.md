# NBI Klaviyo API Integration

Headless integration for connecting to and interacting with the Klaviyo API for the NBI project.

## Setup

1. Create a virtual environment:
   ```sh
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

2. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```

3. Configure credentials (see `.env.example` for required variables).

## Usage

```sh
python main.py
```

## Directory Structure

- `input/` — Input data files
- `output/` — Generated output files
